# amosdavis.github.io

My personal GitHub Pages site built with Jekyll.

## Local Development

To run locally:

```bash
bundle install
bundle exec jekyll serve
```

Then visit `http://localhost:4000`

## Adding Posts

Create new posts in `_posts/` with the filename format: `YYYY-MM-DD-title.md`

## Customization

- Edit `_config.yml` to change site settings
- Edit `about.md` to update the About page
- The site uses the [Minima](https://github.com/jekyll/minima) theme
